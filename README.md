# AutoFlow Orchestrator

**Agentic AI for Autonomous Enterprise Workflows**  
*Built for the ET AI Hackathon 2026*

## Problem Statement 2
AutoFlow is a multi-agent system that takes ownership of a complex, multi-step enterprise process (Vendor Onboarding). It detects failures, self-corrects, and completes the job with minimal human involvement — while keeping a glowing, real-time auditable trail of every decision it makes.

## Demo Highlights
1. **Autonomous Execution:** Upload a document and watch the multi-agent system chew through extraction, background checks, and ERP integration.
2. **Auditable Trail:** Real-time visibility into the "thought process" and confidence scores of each agent.
3. **Human-in-the-loop:** Deliberate "Exception Path" demonstrating how the system pauses for human compliance authorization when a high-risk jurisdiction is detected.

## Setup Instructions
1. Clone this repository.
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:3000` in your browser.
